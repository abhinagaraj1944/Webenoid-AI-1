============================================
   WEBENOID AI - Excel Add-in (Testing)
============================================

HOW TO INSTALL:

1. Extract this zip folder to your desktop.
2. Double-click "Install-WebenoidAI.bat" (Approve Admin prompt).
3. Close ALL Excel windows.
4. Reopen Excel.
5. Go to: Insert > My Add-ins > Shared Folder tab.
6. Click "Webenoid AI" and press Add.
7. Done! The AI panel opens on the right side.

============================================

HOW TO UNINSTALL:

1. Double-click "Uninstall-WebenoidAI.bat".
2. Restart Excel.

============================================

NOTE: Requires active internet connection to communicate with the development server.
